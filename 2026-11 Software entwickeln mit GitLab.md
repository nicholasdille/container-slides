# Software entwickeln mit GitLAb

## Beschreibung

GitLab positioniert sein gleichnamiges Produkt als eierlegende Wollmilchsau für alle DevOps-Belange – und das zu Recht.

Dieser Workshop führt praktisch an GitLab heran und stellt grundlegende Konzepte für die Software-Entwicklung anhand eines Beispielprojekts vor. Der Workshop stellt Schritt für Schritt vor, wie eine Pipeline aufgebaut wird und welche Features GitLab dafür anbietet. Es wird das Übersetzen und Testen von Software erlernt, aber auch der Aufbau komplerer Pipelines für die Zusammenarbeit im Team sowie der Rollout in Test- und Produktionsumgebungen.

Im Laufe des Workshops werden zusätzliche und hilfreiche Features von GitLab vorgestellt und das Beispielprojekt stetig erweitert.

## Vorkenntnisse

- Die Konzepte von DevOps und CI/CD müssen bekannt sein.
- Die Nutzung von Werkzeugen für CI/CD (z.B. Jenkins, GitHub, Azure DevOps etc.) ist hilfreich.

## Lernziele

- Umgang mit GitLab
- Anlegen und Konfigurieren von Softwareprojekten
- Einrichten von Pipelines zum Bauen, Testen und Verteilen von Software
- Ausrollen von Software in Test- und Produktionsumgebungen

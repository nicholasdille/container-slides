# Workshop: Observability for Cloud Native

This site contains the exercises to learn about observability.

## Audience

Software developers and system administrators who want to learn observability.

## Duration

One day

## Expected knowledge

Understanding of Kubernetes and container orchestration

## Goals

XXX

## Style

20% theoretical introduction

80% practical exercises

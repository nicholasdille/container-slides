# Einführung in Cluster API

## Beschreibung

Cluster API ermöglicht es, Kubernetes-Cluster in YAML zu beschreiben und daraus auszurollen und zu verwalten. Das bedeutet, dass ein Cluster selbst zur Kubernetes-Ressource wird und Veränderungen durch Operatoren umgesetzt werden. Es handelt sich bei dem Projekt um ein offizielles Projekt der Kubernetes SIGs und besitzt Integrationen für viele Public und Private Cloud-Anbieter.

Dieser Vortrag bietet eine Einführung in die Funktionsweise von Cluster API und demonstriert den Rollout, die Skalierung und die Löschung eines Clusters bei einem Public Cloud-Anbieter.

## Vorkenntnisse

- Gute Kenntnis von Kubernetes inklusive der Konzepte
- Gundliegendes Veständnis von Operatoren
